# Handoff: Редизайн 1M Agency — направление «PRESS»

## Overview
Полный визуальный редизайн внутренней платформы агентства **1M Agency** (управление клиентами, контент-планом, съёмками, задачами, таргетом). Направление **PRESS** — смелая, типографическая, премиально-минималистичная эстетика: near-black фон, кислотно-лаймовый акцент, огромные condensed-заголовки. Основной сценарий использования — **iPhone**, поэтому дизайн mobile-first.

Проработаны 4 экрана + нижняя навигация: **Главная (дашборд)**, **Календарь (контент-план)**, **Задачи (канбан)**, **Профиль**. Поддерживаются **тёмная и светлая** темы.

## About the Design Files
Файлы в папке `reference/` — это **дизайн-референс, собранный в HTML/React** (прототип, показывающий внешний вид и поведение). Это **НЕ продакшн-код для копирования один в один**.

Задача — **воссоздать эти экраны в вашем существующем стеке**:
- **React + Vite**, только **inline-стили** (без Tailwind/CSS-модулей) — как в текущем проекте
- **Supabase** — источник данных (в референсе данные замоканы в `reference/directions/data.js`; подключите реальные запросы)
- **Lucide React** — иконки (в референсе они нарисованы вручную в `lib.jsx` как компонент `Ic`; замените на эквиваленты Lucide — соответствия указаны ниже)

Используйте установленные в проекте паттерны и компоненты. Перенесите **дизайн-токены, типографику, лейаут и поведение**, а структуру данных адаптируйте под реальные таблицы Supabase.

| | Было | Стало (PRESS) |
|---|---|---|
| Заголовки | Bebas Neue | **Anton** (Google Fonts) |

## Interactions & Behavior
- **Переключение вкладок:** мгновенная смена экрана с анимацией входа `pressSwapIn` — `opacity 0→1` + `translateY(10px→0)`, 0.32s, `cubic-bezier(.2,.7,.2,1)`. Уважать `prefers-reduced-motion`.
- **Календарь:** тап по дню → выбор дня (`selected`), повестка снизу обновляется. По умолчанию выбран `today` (13).
- **Тоггл «Тёмная тема»** в Профиле переключает тему всего приложения.
- **Чипы фильтра** в календаре — выбор клиента (в референсе статичны; подключите фильтрацию).
- **Кнопки навигации месяца** `‹ ›` — переключение месяца (в референсе статичны; подключите).
- **Hover/active (десктоп):** на карточках/кнопках — лёгкое поднятие границы до `--line2` или подложка `--accent` с alpha; задайте по паттернам проекта.

## State Management
- `theme`: `'dark' | 'light'` — глобально (контекст/стор), управляется тогглом в Профиле и переключателем темы.
- `activeTab`: `'home' | 'cal' | 'tasks' | 'me'` — для мобильной навигации (на десктопе — роутинг по существующим путям `/`, `/calendar`, `/tasks` и т.д.).
- `selectedDay` (Календарь): число; по умолчанию = сегодня.
- `push` (Профиль): bool для тоггла уведомлений.
- **Данные** (заменить мок на Supabase): метрики дашборда, список съёмок, прогресс клиентов, посты по дням месяца (`{ client, type, status }`), задачи по колонкам (`{ title, client, who, type, due }`), профиль и статистика. Точная форма мока — в `reference/directions/data.js`.

## Files (в этом пакете, папка `reference/`)
- `1M Agency - PRESS.html` — запускаемый прототип **мобайл** (открыть в браузере; переключатель темы сверху, навигация по таб-бару).
- `1M Agency - PRESS Desktop.html` — запускаемый прототип **десктоп** (сайдбар + рабочая область).
- `directions/press.jsx` — **главный мобайл-файл**: все 4 экрана + таб-бар + токены тем (объект `c`), палитры типов/статусов.
- `directions/desktop-press.jsx` — **главный десктоп-файл**: сайдбар + Дашборд/Календарь/Задачи + те же токены.
- `directions/press-app.jsx`, `directions/desktop-app.jsx` — обёртки-демо (управление темой).
- `directions/data.js` — мок-данные (форма данных для Supabase).
- `directions/lib.jsx` — иконки (`Ic`) и помощник `buildMonth(year, monthIndex)` для сетки календаря (неделя с понедельника).
- `frames/ios-frame.jsx` — только iPhone-рамка для мобильного превью; **в продакшн не нужна**.
